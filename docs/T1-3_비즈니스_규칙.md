## 명명 규칙

- 규칙 ID: `BIZ-[순번]` (3자리 zero-padding)
- 영역: 해당 규칙이 적용되는 도메인 모듈
- 우선순위: High = 시스템 무결성에 직접 영향, Medium = 기능 정확성에 영향
