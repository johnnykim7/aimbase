## 작성 가이드

1. **작성 대상**: 3개 이상 필드 폼, 동적 필드, 실시간 갱신, 외부 연동 페이지만 기재
2. **단순 CRUD 제외**: 목록 조회 → 상세 → 수정 → 삭제만 있는 페이지는 T3-3으로 충분
3. **진입 조건**: 인증, 권한, 선행 조건 명시
4. **시나리오 테이블**: 단계별 사용자 행동 → 시스템 반응 → API 호출 매핑
