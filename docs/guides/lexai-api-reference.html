공통 사항

Base URL

http://localhost:8080/api/v1

인증

모든 API 요청에는 JWT 토큰을 Authorization 헤더에 포함해야 합니다. /api/v1/auth/login으로 토큰을 발급받으세요.

Authorization: Bearer eyJhbGciOiJIUzI1NiIsInR5cCI6IkpXVCJ9...

테넌트 식별

멀티테넌트 환경에서 요청마다 X-Tenant-Id 헤더를 포함해야 합니다.

X-Tenant-Id: tenant-001

공통 응답 포맷

모든 API는 ApiResponse<T> 래퍼로 응답합니다.

{
  "success": true,
  "data": { ... },
  "error": null,
  "pagination": null
}

페이지네이션 응답

{
  "success": true,
  "data": [ ... ],
  "pagination": {
    "page": 0,
    "size": 20,
    "totalElements": 42,
    "totalPages": 3
  }
}

에러 응답

{
  "success": false,
  "data": null,
  "error": "에러 메시지"
}

Chat Completions API

LLM 채팅 API. 플랫폼의 핵심 엔드포인트로, 다양한 LLM 프로바이더에 통합 인터페이스를 제공합니다. 도구 호출, 멀티모달 입력, 스트리밍, 구조화된 출력을 지원합니다.

POST /api/v1/chat/completions 채팅 완성 요청

Request Body

필드타입필수설명
modelstring선택LLM 모델명 (예: claude-3-5-sonnet-20241022, gpt-4o). 미지정 시 라우팅 규칙 적용
session_idstring선택세션 ID. 미지정 시 자동 생성. 동일 세션에서 대화 이력 유지 (TTL 24h)
messagesarray필수메시지 배열. 각 메시지는 role과 content 포함
streamboolean선택true 시 SSE(Server-Sent Events) 스트리밍 응답
actions_enabledboolean선택액션(쓰기/알림) 실행 허용 여부
connection_idstring선택특정 LLM 연결 지정
tool_filterobject선택도구 필터링 (CR-006)
tool_choicestring선택도구 선택 모드: auto, none, 또는 특정 도구명
response_formatobject선택구조화된 출력 요청 (CR-007)

messages[] 구조

필드타입설명
rolestringuser, assistant, system
contentstring | array텍스트 문자열 또는 멀티모달 파트 배열

멀티모달 content 파트

[
  { "type": "text", "text": "이 이미지를 설명해주세요" },
  { "type": "image_url", "image_url": { "url": "data:image/png;base64,..." } }
]

tool_filter 구조 (CR-006)

필드타입설명
allowed_toolsstring[]허용할 도구 이름 목록 (화이트리스트)
exclude_toolsstring[]제외할 도구 이름 목록 (블랙리스트)

response_format 구조 (CR-007)

필드타입설명
typestringjson_schema
schema_refstring저장된 스키마 참조 (예: order-response/1)
schemaobject인라인 JSON Schema 정의

Response (비스트리밍)

{
  "success": true,
  "data": {
    "id": "chat-uuid",
    "model": "claude-3-5-sonnet-20241022",
    "session_id": "session-uuid",
    "content": [
      { "type": "text", "text": "응답 텍스트" }
    ],
    "actions_executed": [],
    "usage": {
      "input_tokens": 150,
      "output_tokens": 320,
      "cost_usd": 0.0021
    }
  }
}

Response (구조화된 출력, CR-007)

{
  "content": [
    {
      "type": "structured",
      "schema": "order-response/1",
      "data": { "order_id": "ORD-001", "status": "confirmed" }
    }
  ]
}

스트리밍 응답 (SSE)

stream: true 시 text/event-stream으로 응답합니다.

event: delta
data: {"delta": "응답 텍스"}

event: delta
data: {"delta": "트입니다."}

event: done
data: {"done": true}
도구 호출 루프: 오케스트레이터는 LLM 도구 호출을 자동 실행하며, 최대 5회 루프합니다 (BIZ-001). 도구 결과를 다시 LLM에 전달하여 최종 응답을 생성합니다.

메시지, 세션, Redis 캐시를 모두 삭제합니다. 응답 본문 없음 (204).
