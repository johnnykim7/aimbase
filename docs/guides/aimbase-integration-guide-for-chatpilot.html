1. Aimbase가 제공하는 기능

Aimbase는 ChatPilot의 모든 AI 기능을 담당하는 백엔드 AI 엔진입니다. ChatPilot 앱은 Aimbase REST API만 호출하면 됩니다.

🤖 LLM 통합 관리

  • Claude, GPT, Ollama 단일 API
  • 모델 자동 라우팅 / 폴백
  • SSE 스트리밍 응답
  • 구조화 출력 (JSON Schema)
  • 도구 자동 호출 (최대 5회)

📚 RAG 파이프라인

  • 문서 업로드 → 자동 파싱·청킹·임베딩
  • BM25 + 벡터 하이브리드 검색
  • Cross-encoder 재랭킹
  • Contextual Retrieval (검색 정확도 49%↑)
  • Parent-Child 계층 검색
  • 시맨틱 캐시

📝 문서 자동 생성

  • 상담 보고서/FAQ 문서 AI 생성
  • DOCX, PDF, PPTX, XLSX 지원
  • 템플릿 관리 (변수 바인딩)
  • 스키마 기반 문서 생성
  • 테마 프리셋

🛡️ 정책 엔진 (가드레일)

  • PII 자동 마스킹 (Presidio)
  • 금지어 차단 (DENY)
  • 승인 필요 (REQUIRE_APPROVAL)
  • 레이트 리밋 (비용 한도)
  • 감사 로깅 (상담 추적성)

⚙️ 워크플로우 엔진

  • DAG 기반 멀티스텝 자동화
  • LLM 호출 / 도구 실행 / 조건 분기
  • 사람 승인 스텝 (HUMAN_INPUT)
  • 병렬 실행
  • 실행 이력 추적

🎙️ 음성 + 평가

  • STT — 음성 상담 텍스트 변환
  • TTS — 답변 음성 읽어주기
  • RAGAS 품질 평가
  • LLM Judge (환각/독성 탐지)
  • 프롬프트 A/B 비교
멀티테넌시 내장 Database-per-Tenant 아키텍처로 기업별 데이터 완전 격리. 플랜(Starter/Pro/Enterprise)별 API 쿼터 자동 관리.

2. 연동 방식 3가지

A. REST API 직접 호출

ChatPilot BE → Aimbase REST API

가장 간단, 권장
HTTP 호출만으로 모든 AI 기능 사용

B. MCP 서버 래핑

ChatPilot 전용 MCP 서버 구축
→ Aimbase API를 MCP Tool로 노출

AI 에이전트가 도구를 자율 선택할 때

C. Aimbase 내부 MCP 직접 연결

ChatPilot → Python Sidecar MCP

39개 내부 도구를 세밀하게 제어할 때

권장 전략: A 먼저 → 필요 시 B/C 추가 대부분 REST API로 충분합니다. MCP는 "에이전트 자율 도구 선택"이 필요한 고급 시나리오에서 사용.

10. MCP 서버 코드 예시

chatpilot_mcp_server.py
from mcp.server import Server
from mcp.types import Tool, TextContent
import httpx

AIMBASE = "http://localhost:8080"
server = Server("chatpilot-ai")

class AimbaseClient:
    def __init__(self, tenant_id, token):
        self.headers = {
            "Authorization": f"Bearer {token}",
            "X-Tenant-Id": tenant_id,
            "Content-Type": "application/json"
        }

    async def _post(self, path, body):
        async with httpx.AsyncClient(timeout=60) as c:
            r = await c.post(f"{AIMBASE}{path}", json=body, headers=self.headers)
            return r.json()

    async def chat(self, messages, **kw):
        return await self._post("/api/v1/chat/completions", {"messages": messages, **kw})

    async def search(self, query, sources, top_k=5):
        return await self._post("/api/v1/knowledge-sources/search",
            {"query": query, "source_ids": sources, "top_k": top_k, "search_type": "HYBRID"})

    async def gen_doc(self, prompt, fmt="docx"):
        return await self._post("/api/v1/documents/generate-json",
            {"prompt": prompt, "format": fmt})

client = AimbaseClient("smart-commerce", "JWT_TOKEN")

# ──── MCP Tool 등록 ────
@server.tool()
async def answer_with_knowledge(question: str, source_ids: str = "ks-faq") -> list[TextContent]:
    """FAQ/매뉴얼 기반 고객 질문 답변"""
    r = await client.chat([
        {"role":"system", "content":"고객 상담 AI. 친절하고 정확하게 답변."},
        {"role":"user", "content": question}
    ], knowledge_source_ids=source_ids.split(","))
    return [TextContent(type="text", text=str(r["data"]))]

@server.tool()
async def classify_intent(message: str) -> list[TextContent]:
    """고객 문의 의도 분류 (환불/배송/제품문의/불만 등)"""
    r = await client.chat([
        {"role":"system", "content":"고객 문의를 분류. JSON으로 반환."},
        {"role":"user", "content": message}
    ], response_format={"type":"json_schema", "schema_id":"intent-classification"})
    return [TextContent(type="text", text=str(r["data"]))]

@server.tool()
async def search_knowledge(query: str, top_k: int = 5) -> list[TextContent]:
    """FAQ/매뉴얼 지식 검색 — 벡터+키워드 하이브리드"""
    r = await client.search(query, ["ks-faq", "ks-manual"], top_k)
    return [TextContent(type="text", text=str(r["data"]))]

@server.tool()
async def generate_report(report_type: str, data: str) -> list[TextContent]:
    """상담 보고서 생성 (PDF/DOCX)"""
    r = await client.gen_doc(f"{report_type} 보고서 생성:\n{data}")
    return [TextContent(type="text", text=str(r["data"]))]

@server.tool()
async def analyze_sentiment(message: str) -> list[TextContent]:
    """고객 감성 분석 (긍정/부정/중립 + 에스컬레이션 필요 여부)"""
    r = await client.chat([
        {"role":"system", "content":"감성 분석. JSON: {sentiment, score, escalation_needed}"},
        {"role":"user", "content": message}
    ], response_format={"type":"json_schema", "schema_id":"sentiment-analysis"})
    return [TextContent(type="text", text=str(r["data"]))]

@server.tool()
async def summarize_conversation(session_id: str) -> list[TextContent]:
    """상담 이력 요약 (주제, 감성, 해결 여부)"""
    r = await client.chat([
        {"role":"system", "content":"상담 이력 요약. JSON: {topic, sentiment, resolved, summary}"},
        {"role":"user", "content":f"세션 {session_id} 상담 요약"}
    ])
    return [TextContent(type="text", text=str(r["data"]))]

@server.tool()
async def draft_email(customer_name: str, inquiry: str, resolution: str) -> list[TextContent]:
    """고객 답변 이메일 초안 생성"""
    r = await client.gen_doc(
        f"답변 이메일:\n고객: {customer_name}\n문의: {inquiry}\n해결: {resolution}")
    return [TextContent(type="text", text=str(r["data"]))]
