# CR-039 원본 요구사항 — 고급 확장 도구 (TeamCreate/Delete + NotebookEdit + LSPTool)

## 요청일: 2026-04-09
## 요청자: sykim

## 배경
openclaude 전수 비교(2026-04-07)에서 식별된 잔여 갭 12개 중 CR-037(핵심 도구 4개), CR-038(에이전트 자율성 4개) 완료 후 남은 마지막 4개 도구.

## 범위
1. **TeamCreateTool** — Swarm 패턴에서 동적 팀 생성 (에이전트 그룹화 + 역할 배정)
2. **TeamDeleteTool** — 팀 해체 (1번과 세트)
3. **NotebookEditTool** — Jupyter Notebook(.ipynb) 셀 편집 (RAG 평가, 임베딩 드리프트, 벤치마크 시각화)
4. **LSPTool** — Language Server Protocol 클라이언트 (타입 정보, 심볼 참조, 정의 추적)

## 설계 판단
- **Team 모델**: 세션 스코프 휘발성 (Redis 캐시 + DB 기록용). Swarm 패턴 특성상 세션 종료 시 팀 해체
- **LSPTool 범위**: 핵심 3개(definition, references, hover)만 초기 구현. 풀 LSP 클라이언트는 과설계
- **NotebookEdit**: .ipynb는 JSON이므로 Java 직접 편집 (Python sidecar 경유 불필요)

## 기대 효과
- 멀티에이전트 Swarm 패턴 지원으로 복잡한 협업 시나리오 자동화
- Notebook 편집으로 RAG 평가/벤치마크 결과를 재현 가능한 형태로 관리
- LSP 연동으로 에이전트 코드 분석 품질 향상 (타입 추론, 정의 추적)

## 제목 제안
