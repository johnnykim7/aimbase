# CR-038 원본 요구사항 — 에이전트 자율성 강화

## 요청일: 2026-04-08
## 요청자: sykim

## 배경
openclaude 전수 비교(2026-04-07)에서 식별된 잔여 갭 12개 중 에이전트 자율성 관련 4개 도구를 묶어 구현.

## 범위
1. **ListMcpResources** — UI가 아닌 에이전트가 자율적으로 MCP 리소스 탐색
2. **ReadMcpResource** — 에이전트가 MCP 리소스를 직접 읽기
3. **RemoteTriggerTool** — Cron이 아닌 이벤트 기반 즉시 트리거
4. **BriefTool** — 세션 시작 시 이전 작업 요약 즉시 로딩 (메모리 시스템과 용도 다름)

## 기대 효과
- MCP 리소스를 에이전트가 직접 탐색·읽기하여 사람 개입 없이 자율 작업 가능
- 이벤트 기반 즉시 트리거로 Cron(주기적)과 보완적인 자동화 달성
- 세션 브리핑으로 에이전트가 이전 작업 컨텍스트를 즉시 복원

## 제목 제안
"에이전트 자율성 강화 — MCP 리소스 탐색·읽기 + 이벤트 트리거 + 세션 브리핑"
