## 사전 준비

### 환경 기동

```bash
# 1. 인프라 (이미 떠 있으면 스킵)
cd backend
docker-compose up -d postgres-master postgres-tenant-dev redis

# 2. BE 기동 (local 프로파일 → 자동으로 dev 테넌트 + admin 생성)
cd backend/platform-core
../../gradlew bootRun --args='--spring.profiles.active=local'
# → 로그에서 JWT 토큰 복사해둘 것

# 3. FE 기동
cd frontend
npm run dev
# → http://localhost:5173

# 4. Python 사이드카 (RAG 테스트 시 필요)
cd python-sidecar
uvicorn src.rag_pipeline.server:app --host 0.0.0.0 --port 8000
```

### 로그인 정보

| 항목 | 값 |
|------|-----|
| URL (BE) | http://localhost:8080 |
| URL (FE) | http://localhost:5173 |
| Email | admin@dev.local |
| Password | admin123 |
| Tenant | tenant_dev |
