### 예상 질문

| 질문 | 답변 | 시연 가능 |
|------|------|---------|
| 데이터 보안은 어떻게? | Database-per-Tenant로 물리적 격리. PII 마스킹 정책 적용 가능. | ✅ (정책 시뮬레이션) |
| 비용은 어떻게 관리? | 정책으로 Rate Limit, 모니터링으로 실시간 비용 추적. | ✅ (모니터링 화면) |
| 어떤 AI 모델 지원? | OpenAI, Anthropic Claude, Ollama(로컬). 라우팅으로 자동 선택 가능. | ✅ (연결 관리) |
| 기존 시스템과 연동은? | MCP 서버로 도구 연동, Webhook/Slack 알림, PostgreSQL Write 액션. | ✅ (MCP 화면) |
