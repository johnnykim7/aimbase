| **n8n**        | 범용 워크플로우 자동화    | 앱 연동(500+), 스케줄, 병렬 실행       | LLM 전용 기능 전반         |

### Aimbase 고유 차별점 (다른 곳에 없는 것)

1. **Database-per-Tenant 멀티테넌시** — 완전한 데이터 격리
2. **Self-RAG + HyDE** — 검색 품질 자동 평가 + 가설적 문서 생성
3. **SpEL 기반 정책 엔진** — 조건식으로 유연한 규칙 평가
4. **PII 마스킹 6종** — 이메일, 전화, 카드, 주민번호 등
5. **토큰 쿼터 관리** — 테넌트별 사용량 제한
