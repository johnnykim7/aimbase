## 12. FlowGuard Agent 범용 도구 확장 (CR-017)

> FlowGuard Agent(flowguard-agent.jar)에 범용 로컬 도구를 추가. 기존 Playwright + WebSocket 구조를 유지하면서 taskType 분기로 확장. AI 오케스트레이션이 아닌 도구 수준 실행 — Aimbase 경유 불필요.

#### PRD-163 | Agent 범용 도구 프레임워크
- **모듈**: FlowGuard > Agent > Core
- **설명**: WebSocket ASSIGN_TASK 프로토콜에 `taskType` 분기를 추가한다. `"playwright"` (기존 동작) 또는 `"tool"` (범용 도구). ToolDispatcher가 toolName으로 적절한 실행기에 위임한다.
- **입력**: ASSIGN_TASK 메시지 (`taskType`, `toolName`, `toolParams`)
- **출력**: TASK_RESULT 메시지 (실행 결과)
- **제약조건**:
  - `agent-config.json`으로 allowedPaths, allowedCommands, 타임아웃 설정
  - PathValidator: 경로 화이트리스트, 심볼릭 링크 추적 차단
  - CommandSanitizer: 명령 화이트리스트, Docker 위험 플래그 차단
- **우선순위**: High | **MVP**: Y | **상태**: 구현완료
- **관련 CR**: CR-017

#### PRD-164 | Claude CLI 도구
- **모듈**: FlowGuard > Agent > Tool > ClaudeCLI
- **설명**: 로컬 소스코드 대상으로 Claude CLI를 실행한다. 도구 수준 실행이며 AI 오케스트레이션이 아니다. Aimbase 컨테이너 내 CLI(컨테이너 작업용)와는 접근 자원이 다름.
- **입력**: `project_path`, `prompt`, `config_dir` (선택), `model` (선택), `max_turns` (선택)
- **출력**: CLI 실행 결과 (stdout, stderr, exit_code)
- **제약조건**:
